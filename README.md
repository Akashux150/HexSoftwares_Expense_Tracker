a# 💰 Expense Tracker 

## 📌 Project Overview

This project is a **Command-Line Expense Tracker** built using Python.
It allows users to record, view, and manage their daily expenses efficiently using a CSV file for data storage.

---

## 🚀 Features

* ➕ Add new expenses with date, category, description, and amount
* 📋 View all recorded expenses
* 💰 Calculate total expenses
* 💾 Data stored persistently in a CSV file
* 🧾 Simple and user-friendly CLI interface

---

## 🛠️ Technologies Used

* Python 🐍
* CSV Module (for file handling)
* OS Module

---

---

## ▶️ How It Works

1. Select an option from the menu:

   * Add Expense
   * View Expenses
   * Expense Summary
2. Enter details like:

   * Date (YYYY-MM-DD)
   * Category
   * Description
   * Amount
3. Data is saved automatically in `expenses.csv`
4. View all expenses anytime
5. Get total expense summary instantly

---

## 📸 Output

* Displays expenses in terminal
* Stores data in CSV file
* Shows total expenses calculation
